# UX APEX // 10-Week Advanced Design Roadmap Tracker

An interactive, premium client-side web application to track progress through the 10-Week Advanced UX Design Roadmap syllabus.

## 🌟 Features

- **Dynamic Interactive Syllabus**: 10 weeks of study goals and practice guidelines with progress tracking.
- **Client-Side Storage**: No database required. Complete state preservation (`localStorage`) for checklist states, custom notes, and sandbox sub-tasks.
- **Focus Studio**:
  - Distraction-free active task display.
  - Pomodoro timer with presets (25m Focus, 50m Deep Work, 5m Break, 15m Break).
  - Ambient Sound Mixer (Synthesized Rain, White Noise, and Binaural Beats) powered by the Web Audio API.
  - Guided Box Breathing tool.
- **Stats Dashboard**: Analytics on study completion rate, streak logs, data backup, and data resets.
- **Aesthetic Themes**: Includes Deep Space (Neon Dark), Nordic Clean (Light Minimalist), and Cyberpunk (Neon pink/cyan).

## 🚀 Getting Started

Simply open `index.html` in your browser, or run a local server:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
